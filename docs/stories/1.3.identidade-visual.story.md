# Story 1.3 — Identidade Visual Fundacional (Tipografia, Paleta, Layout Root)

## Status

Ready for Review

## Story

**As a** Geovana (usuária-âncora),
**I want** que toda página do app, mesmo as vazias, já transmita o ethos visual definitivo do produto (tipografia serifada, paleta neutra-quente, espaço em branco generoso),
**so that** desde o primeiro acesso eu reconheça que este é o santuário de escrita que foi construído para mim.

## Acceptance Criteria

1. Fonte serifada primária (Lora como default — fácil de trocar depois pós-teste com Geovana) carregada via `next/font` self-hosted com performance otimizada.
2. Tokens de cor definidos via CSS custom properties em `app/globals.css`: background papel (`#FAF7F2`), texto preto suave (`#1A1A1A`), acentos sépia/borgonha discretos.
3. `app/layout.tsx` configurado com fonte primária aplicada globalmente, background neutro-quente, meta tags básicas (já feito na Story 1.1, ajustar fonte).
4. Componente `<Container>` reutilizável em `components/layout/Container.tsx` definindo largura máxima de leitura (~680px), centralização horizontal e padding lateral responsivo.
5. Tipografia base estilizada: `h1`-`h4`, parágrafo, link, blockquote — com hierarquia visual elegante e consistente.
6. Comportamento `prefers-reduced-motion` respeitado em qualquer transição CSS aplicada no layout root.
7. Página inicial `app/page.tsx` atualizada para renderizar **"Escritos da Geo"** centralizado em tipografia primária, com largura de leitura, em fundo neutro-quente — esteticamente acabado (não mais texto cru).
8. Lighthouse score nas categorias Performance, Accessibility, Best Practices ≥ 90 na página inicial vazia (SEO pode ficar abaixo nesta etapa).

### Decisões técnicas

- **Lora como default**: melhor compromisso entre legibilidade serifada e neutralidade (vs Fraunces mais autoral, EB Garamond mais clássico, Source Serif mais técnico). Trocável em 1 token caso Geovana prefira outra.
- **Sem dark mode no MVP**: doutrina diurna da escrita íntima. `prefers-color-scheme` ignorado deliberadamente.
- **`text-wrap: balance` em headings** e **`text-wrap: pretty` em parágrafos**: features modernas que melhoram tipografia sem JS.
- **`clamp()` em font-size**: responsividade fluida sem media queries; mobile fica adequadamente menor sem precisar de breakpoints.
- **Acentos sépia**: usados com parcimônia (só links, focus, selection) para não competir com o conteúdo.
